# NotebookLM 接入 Hermes Agent，把谷歌知识库变成私人外挂

## 来源

公众号：元小二学AI  
原文链接：https://mp.weixin.qq.com/s/UEm9ZCLX9eme4-cWbh1ReQ

## 核心功能

Hermes 无缝集成 NotebookLM，带来以下能力：

| 功能 | 说明 |
|------|------|
| 自然语言操控 | 中文指令直接操作 NotebookLM，无需打开浏览器 |
| 一句话生成幻灯片 | 自动生成 PPT，保存到本地 |
| 自动生成讲解视频 | 20分钟生成真人口播风格视频 |
| 极速检索 | 自动优化查询词，返回完整答案 |
| 多设备同步 | 云端存储，随时访问 |

## 安装方式

核心开源项目：**notebooklm-py**  
GitHub：https://github.com/win4r/notebooklm-py

在 Hermes 里输入："阅读这个项目的安装说明并安装"，然后粘贴仓库链接，让 Hermes 自己安装自己。

## 可用场景

| 场景 | 操作 |
|------|------|
| 论文研究 | 搜索论文，自动创建 Notebook 并添加 |
| 深度学习 | 让 Hermes 通过 NotebookLM 讲解技术点 |
| 内容创作 | 一句话生成播客、视频、PPT |
| 知识整理 | 自动生成 Mind Map |
| 自我测验 | 生成 Quiz 检验理解 |

## 关键价值

- 解决传统 RAG 的 Token 消耗和维护成本问题
- 所有知识系统化存入 NotebookLM
- Hermes 作为私人知识管家，随时调用、生成、分享
- 长期积累形成知识飞轮

## 个人评价

作者认为这是目前最接近"零门槛"的个人知识库方案，适合想把 AI 变成个人系统一部分的用户。
