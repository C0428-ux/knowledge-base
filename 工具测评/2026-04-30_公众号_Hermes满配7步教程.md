# Hermes 满配 7 步教程

## 核心观点

裸装 Hermes 和满配 Hermes 是两个物种。满配版：长期记忆、全网抓取、语音图片生成、Token 消耗更少。

---

## 第一步：写 SOUL.md，定义它是谁

Hermes 裸装后不知道你是谁、做什么工作风格。每次对话都要重新交代。

**解决方案**：写 SOUL.md，定义人格和角色。

**资源**：[agency-agents-zh](https://github.com/jnMetaCode/agency-agents-zh) — 211 个中文角色模板，18 个部门分类，覆盖工程、设计、营销、产品、金融、HR 等。

找到需要的角色，复制到 SOUL.md 激活。

---

## 第二步：换掉内置记忆，装 Hindsight

Hermes 内置 MEMORY.md 硬上限约 2200 字符，只有它"觉得重要"时才写入，跨会话记忆基本不行。

**Hindsight** 的优势：
- 自动从每轮对话提取实体、事实、关系、时间戳，建立知识图谱
- 开新会话时，调用 AI 前自动注入相关记忆

**配置方法**：
```
hermes memory setup
# 选择 hindsight，向导自动搞定
```

然后去 https://ui.hindsight.vectorize.io/connect 注册拿免费 API Key，验证：
```
hermes memory status
```
看到 auto-recall 和 auto-retain 亮了即可。

---

## 第三步：装抓取工具，读懂整个互联网

四个工具：
- **Jina Reader** — 单页抓取，快
- **Crawl4 AI** — 批量深度抓取，强
- **Scrapling** — 反爬绕过，稳
- **CamoFox** — 隐身浏览器，专治登录墙

CamoFox 和 Scrapling 可通过 `hermes tools + pip` 启用。

---

## 第四步：搜索 + 文档处理

- **Tavily** — AI 专用搜索，每月 1000 次免费，质量远超普通搜索
- **DuckDuckGo** — 零成本兜底
- **Pandoc** — 万能格式转换器，Word/PDF/Markdown 互转
- **Marker** — PDF 转 Markdown 精度增强版

装完 Marker 后，高精度提取 PDF 内容。

---

## 第五步：表达能力工具链

- **Whisper** — 语音识别，99+ 语言，本地运行，不上传云端
- **Edge TTS** — 语音合成，免费用
- **Fal.ai + FLUX Skill** — 图片生成，高质量出图

装完可对着 Hermes 说话，它回你、给你画图，全程不换工具。

---

## 第六步：精细管控 Token

**RTK（Rust Token Killer）** — 把终端命令 Token 消耗压掉 60%~90%：

```bash
# 安装
brew install rtk
# 或一键脚本
curl -fsSL https://raw.githubusercontent.com/rtk-ai/rtk/refs/heads/master/install.sh | sh
# 集成到 Hermes
rtk init -g
```

**Tokscale** — 实时监控 Token 消耗：
```bash
npx tokscale@latest
tokscale --hermes  # 只看 Hermes 的消耗
```

**hermes-hudui** — Web UI，14 个 Tab，按模型/会话/工具调用拆解 Token 去向。

## 第七步：认识 Hermes 生态全貌

- **awesome-hermes-agent** — 一站式资源汇总
- **hermes-ecosystem** — 80+ 工具可视化地图
- **wondelai** — 380 个跨平台 Skill 一次性装上
- **awesome-agent-skills** — 1000+ Skill 按需挑选

## 核心原则

> "Hermes 裸装就像买了一台跑车，连轮子都没装全就开上路。"

按这 7 步配置完，你才真正拥有一个 AI Agent，而不是一个会聊天的玩具。

> "Read in, Prompt out" — 工具只是放大器，你脑子里有多少东西，决定 AI 能帮你放大多少。
