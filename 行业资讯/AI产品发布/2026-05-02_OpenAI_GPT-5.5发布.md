# Introducing GPT-5.5

**来源：** OpenAI | **发布时间：** 2026-04-23
**链接：** https://openai.com/index/introducing-gpt-5-5/
**分类：** 新产品/模型发布

---

## 核心内容

GPT-5.5 是 OpenAI 迄今为止最智能、最易用的模型，专为复杂现实工作设计。模型能更快理解用户意图、更独立地完成工作，在代码编写与调试、在线研究、数据分析、文档创建、软件操作等任务上表现出色。

**关键创新：** GPT-5.5 实现了显著的智能提升，同时匹配 GPT-5.4 的每 token 延迟，使用显著更少的 token 完成相同任务。

### Agentic Coding

GPT-5.5 是 OpenAI 最强的 agentic 编程模型：
- Terminal-Bench 2.0: **82.7%** (GPT-5.4: 75.1%)
- Expert-SWE (Internal): **73.1%** (GPT-5.4: 68.5%)

合作伙伴包括 Cursor、Lovable、Cognition、Windsurf、GitHub、JetBrains、Sonar。

### 知识工作

OpenAI 内部 85%+ 员工每周使用 Codex，涉及软件工程、财务、通信、市场、数据科学和产品管理。财务团队用 GPT-5.5 在 2 周内完成了 24,771 份 K-1 税表（共 71,637 页）的审查，比上年快两周。

### 科学研究

GPT-5.5 帮助内部发现了关于 Ramsey 数的新证明，这是组合学中的核心对象。还在 FrontierMath Tier 4 上达到 **35.4%** (GPT-5.4: 27.1%)。

## 关键要点
- GPT-5.5 是 OpenAI 迄今最智能模型，在编码、科学推理、数学基准上显著超越 GPT-5.4
- 内部使用率超过 85%，在企业知识工作场景效果显著
- 在 FrontierMath Tier 4 上达到 35.4%，超越 Claude Opus 4.7 的 22.9%
- 已发现组合学中一个新的 Ramsey 数渐近事实证明，并通过了 Lean 验证
