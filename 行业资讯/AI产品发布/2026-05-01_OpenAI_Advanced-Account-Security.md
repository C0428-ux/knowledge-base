# OpenAI 推出高级账户安全功能

## 来源
- **公司**：OpenAI
- **链接**：https://openai.com/index/advanced-account-security
- **发布时间**：2026年4月30日

## 核心内容

OpenAI 推出"高级账户安全"（Advanced Account Security）功能，这是面向高风险用户的可选安全设置，旨在防止数字攻击和账户被盗。

### 目标用户
- 记者
- 民选官员
- 政治异议人士
- 研究人员
- 安全意识强的用户

### 核心功能
1. **更强登录方式**：强制使用 Passkey 或物理安全钥匙，禁用密码登录
2. **更安全账户恢复**：禁用邮件和短信恢复，需使用备份 Passkey、安全钥匙和恢复密钥
3. **更短会话**：缩短登录会话以降低风险，支持登录警报和会话管理
4. **自动排除训练**：高级账户的对话不会用于模型训练

### Yubico 合作
OpenAI 与 Yubico 合作，为用户提供安全钥匙优惠价。

### 关键时间
- 2026年4月30日：功能上线开放注册
- 2026年6月1日：网络安全可信访问成员必须启用

## 关键要点
- OpenAI 推出针对高风险用户的钓鱼攻击防护功能
- 强制使用 Passkey/安全钥匙，禁用传统密码登录
- 该功能是 OpenAI 更广泛网络安全行动计划的一部分
- 企业版功能也在计划中
