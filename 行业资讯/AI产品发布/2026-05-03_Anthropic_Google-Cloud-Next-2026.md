## 核心内容

Anthropic 携 Claude 亮相 Google Cloud Next 2026（4月22-25日，拉斯维加斯），围绕 **"企业级长时运行 AI Agent"** 主题展开一系列分享。

### 主要演讲内容

**1. 真正的多 Agent 系统设计（4月22日 11:15 AM PST）**
- 多 Agent 架构适用场景：上下文隔离、并行执行、专业化
- 四大架构模式及其权衡
- 以上下文为中心的设计分解策略
- 验证子 Agent 和常见陷阱

**2. 前沿 Agent 评估方法解密（4月22日 1:00 PM PST）**
- 如何构建可靠的 Agent 评估体系
- 选择合适的 grader
- 能力测试 vs 回归测试的平衡
- 建立可信赖的评估套件

**3. 后软件时代：Anthropic 对企业 AI 下一时代的愿景（4月22日 2:45 PM PST）**
- 核心观点："未来两年企业软件的变化将超过过去二十年"
- 传统自上而下的规划正在让位于 AI Agent 驱动的自下而上执行
- 现场演示 Vertex AI 上自主定义、构建和迭代的 Agent

**4. Shopify Sidekick 案例研究（4月22日 3:45 PM PST）**
- 如何在 Vertex AI 上实现多步骤工作流执行
- 模型更新在24小时内部署的能力

**5. 长时运行 Agent：企业前沿实践（4月23日 12:00 PM PST）**
- AI Agent 已在生产环境中运行数小时
- 应用于代码库现代化和需求到应用转化的框架

### 客户案例

**Palo Alto Networks**
- 功能开发速度提升 20-30%
- 3,500名开发者快速上手
- 初级开发者任务完成速度提升 70%

**其他合作客户**：Replit、Shopify、Augment Code
