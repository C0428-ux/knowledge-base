# Building the Compute Infrastructure for the Intelligence Age

**链接：** https://openai.com/index/building-the-compute-infrastructure-for-the-intelligence-age

## 核心内容

Stargate 是 OpenAI 长期计算基础设施建设项目，旨在为 AGI 时代提供算力支撑。项目已提前完成 2029 年 10GW 目标，过去 90 天新增 3GW 容量。Abilene（德克萨斯州）为中心站点，GPT-5.5 在此训练完成。

关键信息：
- 基础设施采用 Oracle Cloud + NVIDIA GB200 闭式循环冷却系统
- 水资源管理：每栋建筑初始填充约 2 个奥运泳池水量，满负荷运营年耗量相当于约 4 户普通家庭
- 合作伙伴模式：Vantage Data Centers、Oracle 等多家企业共建
- 社区共建：与 NABTU 工会合作创造就业岗位，向 Port Washington-Saukville 教育基金会捐款

## 关键要点
- OpenAI Stargate 项目已超越 2029 年 10GW 目标，提前完成
- 过去 90 天新增 3GW 算力，体现快速扩展能力
- GPT-5.5 在 Abilene 站点完成训练，代表该基础设施的实际产出
- 计算飞轮效应：更多算力 → 更好模型 → 更低成本 → 更多用户 → 再投资
- 基础设施以「社区共建、利益共享」为原则，包括就业、供水、当地税收等
