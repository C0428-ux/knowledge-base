# Building the Compute Infrastructure for the Intelligence Age

## 来源
- **公司**：OpenAI
- **链接**：https://openai.com/index/building-the-compute-infrastructure-for-the-intelligence-age
- **发布时间**：2026-04-29
- **分类**：平台更新

## 核心内容

OpenAI 的 Stargate 项目已超越原定 2029 年实现 10GW 美国算力的目标，仅用一年多就实现了这一里程碑。过去 90 天内新增 3GW 容量。Abilene, Texas 数据中心使用 Oracle Cloud Infrastructure 和 NVIDIA GB200 系统，GPT-5.5 在此训练。

Stargate 联合芯片商、云计算商、数据中心运营商、能源公司、建筑商、工会等合作伙伴。Abilene 数据中心采用闭式循环冷却，水使用量相当于 4 户普通家庭。

## 关键要点
- 2025 年 1 月设定的 10GW 目标已提前实现
- 过去 90 天新增 3GW 算力
- Abilene 数据中心使用 Oracle Cloud Infrastructure + NVIDIA GB200
- GPT-5.5 在 Abilene Stargate 站点训练
- 与 NABTU 工会合作创造就业岗位
- 水使用量：满负荷运行相当于 4 户普通家庭
