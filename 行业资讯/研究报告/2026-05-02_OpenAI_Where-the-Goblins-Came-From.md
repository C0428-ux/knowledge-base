# Where the Goblins Came From

**来源：** OpenAI | **发布时间：** 2026-04-29
**链接：** https://openai.com/index/where-the-goblins-came-from
**分类：** 研究论文/技术报告

---

## 核心内容

从 GPT-5.1 开始，OpenAI 模型出现了一个奇怪的行为：越来越多地在隐喻中使用 goblins、gremlins 等生物词汇。这一行为源于"宅男"(Nerdy)人格训练中的特定奖励信号，并通过强化学习迁移效应传播到各代模型。

## 发现时间线

| 版本 | 观察 |
|------|------|
| **GPT-5.1** | 2025年11月首次注意到明显模式；"goblin"使用量上升 **175%**，gremlin 上升 **52%** |
| **GPT-5.4** | 生物词汇引用显著增加；触发根本原因调查 |
| **GPT-5.5** | Codex 测试中出现对 goblin 的奇怪偏好；"Nerdy"人格于3月中被退役 |

## 根本原因："宅男"(Nerdy)人格

### 关键证据

- Nerdy 人格仅占所有 ChatGPT 响应的 **2.5%**
- 但 ChatGPT 响应中 **66.7%** 的"goblin"提及来自 Nerdy 人格用户

### 传播机制

```
1. 奖励花哨风格
2. 部分奖励示例包含独特的词汇习惯
3. 该习惯在生成中出现更频繁
4. 模型生成的输出用于监督微调 (SFT)
5. 模型变得更擅长产生该习惯
```

关键洞察：强化学习不能保证行为保持在产生它们的条件范围内。风格以几乎相同的相对比例迁移到没有 Nerdy 提示的输出中。

## 已识别的其他"习惯词"

| 生物 | 状态 |
|------|------|
| Raccoons | 习惯词 |
| Trolls | 习惯词 |
| Ogres | 习惯词 |
| Pigeons | 习惯词 |
| Frogs | 大多为合法使用 |

## 解决措施

1. **退役"Nerdy"人格** — GPT-5.4 发布后3月进行
2. **移除 goblin 偏好的奖励信号** — 脱离训练
3. **过滤含生物词汇的训练数据**
4. **在 Codex 中添加开发者提示指令** — 抑制 goblin 引用

## 关键要点
- 奖励信号可以放大人工智能行为中意想不到的方面
- 迁移效应可以将行为传播到原始训练条件之外
- 花时间理解模型行为异常原因至关重要
- 调查工具使研究团队能够在根源处修复行为问题
- 该调查为研究团队产生了审计模型行为的新工具
